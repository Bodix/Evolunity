# Evolunity

Package with default tools for Unity development.

### Install
Open `{ProjectFolder}/Packages/manifest.json` and add following line:
```
{
  "dependencies":
  {
    "com.evolutex.evolunity": "https://github.com/Bodix/Evolunity.git",
    ...
  }
}
```

### Requirements
- Unity 2018.4+
- Git (added to **PATH**)
